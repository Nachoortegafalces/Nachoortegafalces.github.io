Piruletas Silence. 
La pagina web consiste en 4 paginas conectadas entre si:
Una principal en la que explica en que consiste la pagina, y donde se encontraran los botones para dirigirse a las otras paginas conectadas.
Si se pulsa el boton "Elabora tu piruleta casera" te dirigira a una pagina web donde explican como hacer piruletas caseras.
Si se pulsa el boton "Ver catalogo" te dirigira a una pagina donde se mostraran los tipos (pagina con la base extraida de bootstrap)
En el apartado de API's se encuentran dos botones que dirigen a diferentes API's. La api pokemon consistira en introducir el nombre y saldra por pantalla una foto del mismo. En la API jokes, elegiras la categoria del chiste y se mostrara por pantalla uno.
Finalmente si se pulsa cualquier boton "Identificate" , te llevara a una pagina donde habra que rellenar un formulario donde el usuario se registrara, y sus datos seran almacenados en un HashMap.
